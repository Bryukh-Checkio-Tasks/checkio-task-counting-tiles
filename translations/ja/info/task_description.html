ステファンは彼らの新たな氷の基地に正方形の氷のタイルを使って円形の着陸地帯を作るのに助力を求めています。 彼は構築に着手する前に必要となるタイルの数を勘定したいと考えています。

各タイルは1辺が1メートルの正方形です。 あなたは半径がNメートルの円形地帯が必要とする完全な正方形のタイルと部分的なタイルの枚数を計算する必要があります。 円の中央は4枚のタイルが成す交点とします。例：半径2メートルの円は4枚の完全なタイルと12枚の部分的なタイルが必要です。

入力: 円の半径(float型)

出力: 完全なタイルと部分的なタイルの数の2つの整数からなるリスト([完全なタイル, 部分的なタイル])

どのように使われるか: このタスクはシンプルな幾何学の問題です。しかしあなたはこれを建築や地形学に使うことができます。 プロジェクトに必要な材料の量を計算する際に用いることができるでしょう。

Precondition:
0 < radius ≤ 4
